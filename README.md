# BMI Calculator

A simple and responsive **BMI (Body Mass Index) Calculator** built using **HTML, CSS, and JavaScript**.

## Live Demo
https://bmi-calculator-six-zeta.vercel.app/

## Features
- Calculate BMI instantly
- Simple and clean user interface
- Responsive design
- Fast and lightweight

## Technologies Used
- HTML
- CSS
- JavaScript

## How It Works
1. Enter your height and weight
2. Click the **Calculate** button
3. The application calculates your BMI
4. The result is displayed instantly

## BMI Formula
BMI = Weight (kg) / (Height (m) × Height (m))
